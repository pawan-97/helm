# HELM

## HELM basic commands

* List of availabe helm repo

```
helm repo list
```
* Search Helm Charts

```
helm search repo mysql
```

* Installing a Helm Chart

```
helm install my-mysql stable/mysql
```

* Adding a HELM repo

```
helm repo add <Repo_LInk>
```

* Updating the repo

```
helm repo updatee
```
